# 高效运维社区的DevOps部署流水线演示项目

## 代码
代码管理采用Gitlab，分支模型选择TBD，基于问题创建分支


## 提交阶段（Commit Stage）
在特性分支有提交时触发，执行编译、单元测试任务

## 验证阶段（Integration Stage）
在接受Merge Request之后触发，执行完整的编译、打包、部署、自动化测试、发布过程

## 部署阶段（Deploy Stage）
完全自助式的选择版本进行发布
